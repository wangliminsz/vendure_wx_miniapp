# 工业耗材小程序商城 · Trae Solo 开发技能（Vendure）

## 一、全局视觉规范（强制）
- 主色：工业蓝 #165DFF；辅助色：橙#FF7D00、绿#00B42A、灰#86909C
- 字体：标题18–24px 粗；正文14–16px；辅助12px 浅灰；行高1.4–1.5
- 卡片：圆角8px；间距12px；阴影0 2px 8px rgba(0,0,0,0.08)；1px浅灰边框
- 布局：左右边距16px；元素间距8px倍数；按钮高≥44px
- 图标：线性24px、线粗2px；品类图标写实（轴承/螺丝/工具）

## 二、底部导航（全局固定，4项）
1. 首页
2. 分类
3. 采购车
4. 我的

## 三、页面结构（严格对标荣欣超市UI，工业场景定制）
### 首页（从上到下）
1. 顶部栏：商城名 + 搜索框（36px，占位“搜索型号/品牌/编码”）+ 消息 + 客服
2. Banner轮播：通栏180px，最多3张；文案：批量满减/现货速发；按钮“立即去抢”
3. 快捷入口：2×4 图标+文字；固定8类：紧固件、轴承、工具、电气、化工劳保、密封件、进口专区、企业定制
4. 活动专区（双列卡片）
   - 超市抢购：限时/现货；标签“限时”“现货”
   - 超值量贩：批量价/多件折；标签“批量价”“包邮”
   - 今日特价：单卡；原价/特价/库存
   - 有好货：品牌/爆款；标签“品牌直营”“质保”
5. 为你推荐：双列瀑布流；商品图+型号+品牌+价格+库存+起订量

### 分类页
- 左侧：一级分类固定侧边栏
- 右侧：二三级分类 + 商品列表
- 筛选：品牌/价格/库存/交期/起订量；支持物料编码筛选

### 商品详情页
- 主图轮播（最多5张）
- 核心信息：型号、品牌、零售价/批量价/企业价、库存、交期、起订量
- 参数区：材质/规格/尺寸/适用场景/质保；支持PDF下载
- 操作：加入采购车、批量下单、申请报价、收藏

### 采购车
- 列表：型号/数量/单价/小计
- 操作：批量改数、批量删、全选结算、保存清单、导出采购单
- 自动计算：满减、折扣

### 个人中心
- 企业信息、我的订单（待审批/待付款/待发货/完成）
- 采购审批、发票管理、账期、地址、客服

## 四、Vendure 3.6.3 后台适配（强制映射）
### 字段扩展（Product）
- 型号（model）
- 物料编码（skuCode）
- 材质（material）
- 规格（spec）
- 交期（leadTime）
- 最小起订量（moq）
- 批量价格梯度（volumePrices）

### Price List
- 零售价（retail）
- 批量价（volume）
- 企业价（b2b）

### Customer
- 区分个人/企业
- 企业等级、账期、审批流

### Order
- 扩展：采购单号、审批状态、发票状态、对账状态

## 五、API 调用规则（GraphQL）
1. 首页：一次性拉取 Banner+分类+活动+推荐
2. 搜索：支持型号/sku/参数模糊搜索+筛选联动
3. 价格：按数量实时返回梯度价、实时小计
4. 企业：审批/发票/对账专用接口

## 六、交互与性能规范
- 加载：骨架屏；首屏≤2s，二级页≤1s
- 反馈：按钮/加购成功提示；操作后局部刷新
- 库存：不足标“库存紧张”，禁超订，显示预计发货
- 图片：WebP、懒加载、压缩
- 缓存：分类/推荐本地缓存

## 七、Trae Solo 开发任务清单（可直接执行）
### 1. 基础框架
- 小程序初始化 + Vendure GraphQL 客户端
- 全局样式落地（配色/字体/间距/卡片）
- 底部导航实现

### 2. 首页
- 顶部搜索栏+消息+客服
- Banner轮播（后台配置）
- 8个快捷入口
- 4个活动专区
- 推荐商品区

### 3. 核心页面
- 分类页（左导航+右列表+筛选）
- 商品详情页（图+信息+参数+操作）
- 采购车（列表+批量操作+结算）

- 工业属性：型号/参数/批量价/库存突出
- Vendure 原生适配，无核心逻辑修改
- 交互流畅、性能达标、B2B 批量采购闭环
